# stream-cli-player
Windows only
A webview embedded stream player cli

Usage
```
stream <channel name>
```

Usage (Attached to console)
```
player <channel name>
```

Building
```
git clone https://github.com/shadiy/stream-cli-player.git
cd stream-cli-player
mkdir build
cd build
cmake ..
cmake --build . --config Release
cd Release
stream <channel name>
```
